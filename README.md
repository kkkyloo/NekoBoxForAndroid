Форк NekoBox For Android (база 1.4.2) с фиксами утечек трафика/DNS, ру-правилами маршрутизации,
поддержкой современных протоколов и упрощённым интерфейсом.

## 📥 Скачать
[GitHub Releases](https://github.com/kkkyloo/NekoBoxForAndroid/releases) — берите APK **`arm64-v8a`** для
большинства телефонов (или `armeabi-v7a` для очень старых). x86 не собираем — он нужен только эмуляторам.

## ✨ Что нового в этом форке

- **🌐 Умный авто-выбор (Auto-URL)** — баннер прямо вверху главного экрана. Включаешь — приложение само
  держит самый быстрый сервер из всех твоих подписок (включая sing-box/xray) и переключает, если текущий
  лёг. Баннер подсвечивается, когда режим активен.
- **📊 Живой пинг** — пока открыт список серверов, задержка (мс) **обновляется сама** каждые N секунд, без
  всплывающих окон. Интервал и таймаут настраиваются.
- **⚡ Мгновенный импорт подписки** — вставил ссылку → импортировалось сразу, без лишних окон.
  Поддерживаются ссылки Base64, sing-box, Clash, а также «грязные» подписки с комментариями.
- **🧹 Упрощённые настройки** — сверху только нужное (**⭐ Основное**), всё остальное спрятано под
  переключатель **«Показать расширенные настройки»**.
- **🔌 Новые протоколы** (из слияния с upstream): Juicity, Snell, ShadowsocksR, xhttp-транспорт и др.
- **🔒 Безопасность РФ сохранена** (см. ниже): в режиме VPN локальный порт закрыт, туннель изолирован.

> ⚠️ **При обновлении с ранней версии форка** база данных профилей пересоздаётся с нуля (так надёжнее
> после слияния схем). Просто **добавьте подписку заново** после первого запуска — это разовое действие.

## 🚀 Быстрый старт

1. **Обновите базы маршрутизации:** *Меню → Маршруты → ⋮ → Управление ресурсами* → скачайте `geoip.db` и
   `geosite.db`. *(Приложение само напомнит при первом запуске.)*
2. **Добавьте подписку:** скопируйте ссылку → на главном экране **«+» → «Импорт из буфера обмена»**.
   Импортируется сразу, окно настроек не появляется. Отредактировать группу (фильтры и т.п.) можно
   позже — **долгим тапом** по ней.
3. **Выберите сервер:**
   - **Вручную:** тапните по серверу в списке.
   - **Умный авто-выбор:** включите **баннер «🌐 Умный авто-выбор»** вверху экрана — дальше приложение
     само держит самый быстрый сервер.
4. **Подключитесь:** большая круглая кнопка внизу → подтвердите запрос Android на VPN.
5. **Ограничьте VPN нужными приложениями (рекомендуется):** *Настройки → ⭐ Основное → «Приложения для
   VPN»* → выберите только те приложения, которым нужен обход. Остальные пойдут напрямую.

### Тонкая настройка Auto-URL
*Настройки → 🌐 Умный автовыбор (Auto-URL):*
- **URL для тестирования задержки** (по умолчанию Cloudflare).
- **Страны** — диалог с галочками: страны определяются **автоматически по флагам** в именах
  серверов (🇺🇸🇳🇱🇩🇪). По умолчанию из авто-выбора исключена 🇷🇺 Россия.
- **Подписки** — диалог с галочками: какие подписки участвуют в авто-выборе (если их несколько).
- **Авто-обновление пинга** + **интервал (сек)** и **таймаут пинга (мс)**.

## 🛡 Ключевые изменения безопасности (RKN Hardening)

### 1. Строгая маршрутизация и изоляция туннеля
В оригинальных клиентах приложения, добавленные в исключения (Bypass), могли направлять трафик через
туннель и выявлять IP VPN-сервера.
* В ядре (sing-box) принудительно включена функция `find_process`.
* Трафик исключённых приложений строго фильтруется по UID, предотвращая случайные утечки IP.
* *Тест: `curl 2ip.io --interface tun0` в Termux завершится ошибкой, если приложению не выдан доступ к VPN.*

### 2. Безопасность локального SOCKS5/HTTP прокси
* В режиме VPN локальный прокси-порт **не открывается** (закрытие уязвимости обхода TUN-интерфейса).
* Если локальный прокси всё же включён — слушает только `127.0.0.1` и требует уникальный
  криптографический пароль. Любые inbound, протащенные через кастомный конфиг, вырезаются.

### 3. Защита от подмены IP
* `sniff` и `sniff_override_destination` включены по умолчанию.
  Подробнее — https://4pda.to/forum/index.php?showtopic=1120825&view=findpost&p=142912913

## 🇷🇺 Адаптация для РФ и стабильность

* Предустановлены правила прямого доступа (Direct) к зонам `.ru`, `.рф`, `.su`.
* Диапазон российских IP (`geoip:ru`) автоматически идёт в обход VPN.
* Тест подключения (Ping) переведён на HTTPS.
* При запуске VPN без баз `geosite`/`geoip` приложение не падает, а предлагает их скачать.
